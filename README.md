## 功能介紹

#### 語法

```
# text                          :   標題
1. text                         :   有序清單
-  text                         :   無序清單
- [] item                       :   代辦事項
> text                          :   引用
**text**                        :   粗體
*text*                          :   斜體
~~text~~                        :   刪除線
H~2~O                           :   下標
==text==                        :   標記文字
- text =                        :   分隔線
[link text](https:// "title")   :   超連結
![image alt](https:// "title")  :   圖片
```code```                      :   程式碼區塊

:::info text 
text                            :   區塊(info,warning,success,danger)
:::

```
